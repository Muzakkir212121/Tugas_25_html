Mengenal Simentic Elements Di HTML5

  • Artikel

    • Ketika kita ingin membuat layout untuk menampilkan blok artikel-artikel yang ada di website kita,Tag artikel kita bisa gunakan dengan element dan footer. Karena tag article memiliki type block, Artinya element memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini kita bisa gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel.Karena aside memiliki type/ sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
